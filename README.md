# what is this?

A perfect shadowing

# installation

`npm i shadowing --save`


Then...

```
import {shadowing} from 'shadowing;
shadowing({
      shadow_type:'soft',
      padding:false
});
```

## options

shadowing supports two options :

* *shadow_type* - _hard | soft_ (Defaults to 'soft')
* *padding* - _boolean_ (Defaults to false)
